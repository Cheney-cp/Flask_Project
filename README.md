
##  网络云相册项目说明
    开发环境:
        python3.6
        flask
        mysql
        
  
    所具备功能(参考类似产品http://www.clantu.com):
       用户账号模块:
             1.注册
             2.登录
             3.退出登录
             4.修改密码
             5.修改用户名
       相册功能模块:
             1.上传照片
             2.下载照片
             3.删除照片
             4.删除相册
             5.创建相册
             6.查看相册内容
